# Armies Documentation — AI Poster Manifest

Generated by the art direction team for the Armies documentation site.
These prompts are for Midjourney v6 or DALL-E 3. Generate at 1024×512 (wide/panoramic)
or 512×768 (portrait) depending on placement notes. After generation, save images to
`docs/assets/posters/` with the filename indicated.

**Style baseline for all prompts**: WWII American war propaganda poster style,
bold graphic design, flat areas of color, strong silhouettes, period typography
(condensed sans-serifs, all-caps), visible printing texture, aged paper feel, 1940s.
No photorealistic faces. Silhouettes and symbolic figures only.

## Generation Instructions

1. Use Midjourney v6 with `--ar 16:9` for wide/panoramic posters (width 900px placement)
2. Use `--ar 1:2` for portrait format posters (width 350–400px placement)
3. Use `--ar 7:4` for medium-width posters (width 700px placement)
4. Add `--style raw` for less AI interpretation and more faithfulness to the prompt
5. Add `--stylize 750` for stronger artistic interpretation of the period style
6. After generation, add `--upscale` for the highest resolution
7. Save as PNG to `docs/assets/posters/[filename]` with the filename shown in each entry
8. No further editing should be needed — prompts are designed to work directly

**Tone check**: Each poster should pass the "history nerd pauses scrolling" test. If it looks generic, iterate on the style reference in the prompt with more specificity.
